## üì¶ Quick Start

### DNS Settings

Set your DNS to:

- **Primary:** `8.8.8.8`  
- **Secondary:** `8.8.4.4`

### Jar Creation

```bash
mvn clean package
```
After that, you can run:

```bash
java -jar <your_jar_file>
```

### Launch exe 
Simply start the .exe file. 

### Download videos
All files are downloaded into the **out** folder. Enjoy :-)
